title: Operator Guide Overview
description: How to run, verify, and recover Otto safely.
---

The operator guide is task-oriented. Use it when executing or troubleshooting
real workflows.

Planned guide structure:

- Setup and environment baseline.
- Start/restart/stop operational runbooks.
- Health checks and diagnostics.
- Incident response and recovery checklists.

Operator procedures should always include verification commands and expected
signals.
